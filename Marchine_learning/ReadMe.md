机器学习--Python
================
机器学习是什么？<br>
- 机器学习就是从*数据*中自动分析获取*规律(模型)*，并利用规律对*未知数据进行预测*  

为什么需要机器学习<br>
- 解放生产力
- 解决专业问题
- 提供社会便利  

机器学习在各领域价值？<br>
- 让机器学习程序替换手动步骤，减少企业成本同时提高效率<br>

机器学习知识点：<br>
- 特征工程
- 模型、策略、优化
- 分类、回归、聚类
- Tensorflow
- 神经网络
- 图像识别
- 自然语言处理

机器学习-特征工程
----------------
缺失值，数据转换，重复值是否去重？
sklearn:特征工程工具

特征工程是什么？<br>
特征工程是将*原始数据转换为更好的代表预测模型的潜在问题的特征*的过程，从而*提高了对未知数据的预测准确性*.<br>
特征工程意义:<br>
直接影响预测结果.<br>
